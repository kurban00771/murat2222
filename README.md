# 🏔️ Экскурсии по Кавказу с Муратом

**Путешествия, которые открывают Кавказ заново**

Добро пожаловать на лендинг авторских туров по самым живописным уголкам Кавказа.  
Здесь вы найдёте проверенные маршруты, захватывающие истории и атмосферу настоящих гор.

---

## 🌿 О проекте

Сайт создан для тех, кто хочет увидеть Кавказ глазами местного проводника.  
Мурат — надёжный гид, который организует индивидуальные поездки, экскурсии и приключения с вниманием к каждой детали.

**Что внутри:**
- 🗺️ **Маршруты** — Кольцо-гора, Красные камни, Малое седло, Долина роз и другие знаковые места.
- 📖 **Истории из путешествий** — живые впечатления, адреналин зиплайна и тишина высоких троп.
- 🤝 **Атмосфера** — индивидуальный подход, комфорт, безопасность и настоящий Кавказ без суеты.
- 📍 **Карта** — интерактивный блок, где отмечены ключевые точки маршрутов.
- 📞 **Контакты** — адрес в Кисловодске, email и телефон с защитой от спам-ботов.

---

## ✨ Особенности

- **Адаптивный дизайн** — сайт отлично смотрится на любых устройствах: от смартфонов до больших экранов.
- **Плавная навигация** — якорные ссылки с мягкой прокруткой к разделам.
- **Анимированное меню** — на мобильных устройствах бургер-меню превращается в крестик.
- **Тёмная карта** — Google Maps с кастомным фильтром для стильного отображения.
- **Антиспам** — email и телефон формируются динамически, чтобы избежать парсинга ботами.

---

## 🛠️ Технологии

- HTML5 (семантическая вёрстка)
- CSS3 (Flexbox, Grid, адаптивные медиа-запросы)
- JavaScript (DOM-манипуляции, плавный скролл, защита контактов)
- Google Maps Embed API
